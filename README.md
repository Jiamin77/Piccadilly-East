# Piccadilly-East
Individual work, 2022.9-2023.5

This project is a master's course that spans two semesters and is divided into three parts. The first part analyzes the background and current situation of the Piccadilly East site. The second part considers the design goals and content. The third part delves deeper into and elaborates on the design goals, and presents the design proposal.
